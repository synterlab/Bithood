The Builder

One person, one idea,
built in public.

Bithood is an indie project. One builder, one idea: GPU compute should work like a spot market, not a cloud subscription. The whole thing started in 2026 out of frustration with how expensive and slow it is to get a GPU when you actually need one.

The setup is simple: node operators list idle capacity, renters pick what they need, and Solana Devnet settles the payment on-chain the moment a session ends. No invoices, no credit card holds, no support ticket to get access. Just a wallet, a transaction, and an SSH key.

Right now Bithood connects 1,200+ GPUs across 12 regions. It is still early. The market is live, the billing works, and new node operators are joining weekly. Everything else is being built in public, one commit at a time.

No team page. No funding announcement. Just working software and a transparent market that gets a little better every week.

Quick Start

Zero to running in three commands.

Everything is CLI-first. The dashboard is optional sugar on top.

01 / Install CLI

Get the Bithood CLI

The CLI handles wallet signing, session management, and SSH key injection. It runs anywhere Node 18+ is available.

# install globally npm install -g @bithood/cli # verify bithood --version
02 / Connect Wallet

Link a Solana wallet

Export your keypair from Phantom or Backpack. The CLI stores it locally and signs transactions on your behalf during a session.

bithood auth --keypair ~/.config/solana/id.json # or use ledger hardware wallet bithood auth --ledger
03 / Rent a GPU

Browse and reserve capacity

Filter by tier, region, and price. Reserve sends a single on-chain transaction. SSH credentials are returned within 90 seconds.

# list available H100s under $4/hr bithood markets --tier h100 --max-price 4 # reserve and get SSH creds bithood rent h100-sxm5-us-east-01
04 / End Session

Billing stops the second you disconnect

Sessions are metered to the second on-chain. No orphaned instances, no surprise invoices. End a session at any time.

# graceful shutdown (flushes checkpoints) bithood end --session ses_3kLmXq # force-terminate immediately bithood end --session ses_3kLmXq --force
